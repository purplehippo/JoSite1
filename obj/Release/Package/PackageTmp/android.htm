Ringsim for Android

Ringsim for Android is an application, available on the Android Market (now known as Google Play), for Android phones using Android 2.3 and above.

It is in the market purely to share with a couple of friends, who may be able to help with suggestions and bugs.

What it does do, is successfully searches the microSIRIL library (MSLib) of files (found at http://www.ringsoft.co.uk/change-ringers/ringing-programs/microsiril/method-libraries/), and display the blueline, allowing a few options (like show/don't show the row numbers, and select a trace bell). Bobs and singles are also shown, but I know a few lead-head groups can cause problems. There are approximately 17,680 methods available (see my little Python script below!!).

Python code to count number of methods in MSLib files, count_mslib_methods.py: 

        import os

        def method_counter():
            path = '...\\MSLIB\\'
            files = os.listdir(path)
            counter = 0
            for file in files:
                f = open(path + file, 'r')
                for line in f:
                    counter = counter + 1
            return counter

        print method_counter()
